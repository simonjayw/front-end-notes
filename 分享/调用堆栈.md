# JavaScript 调用堆栈 #

## JavaScript 引擎

![](../assets/调用堆栈/1.png)

- 内存堆（Memory Heap）：内存分配的地方
- 调用栈（Call Stack）：代码执行的地方。LIFO（后进先出）

> 运行时，可以调用引擎之外的api，即WEB API



## 调用栈 ##

```javascript
function multipy (x, y) {
    return x * y
}
function printSquare (x) {
    const s = multipy(x, x)
    console.log(x)
}
printSquare(5)
```

![](../assets/调用堆栈/2.png)

## 堆栈异常追踪 ##

```javascript
function foo() {
    throw new Error('SessionStack will help you resolve crashes :)');
}
function bar() {
	foo();
}
function start() {
	bar();
}
start();
```

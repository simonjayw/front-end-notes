# CSS #

## @ 规则 ##

- @charset	提示文件使用的字符编码方式
- @import	引入一个css文件，@charset 规则不会被引入
- @media		媒体查询
- @page		分页媒体访问网页时的表现设置
- @counter-style		产生一种数据，用来定义列表项
- @key-frames		产生一种数据，用来定义动画关键帧
- @fontface	定义字体
- @support	检查环境，类似 @media
- @namespace	给内部的 css 选择器全部带上特定的命名空间
